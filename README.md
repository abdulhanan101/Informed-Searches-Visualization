# Dynamic Pathfinding Agent

## Description
This project implements a dynamic grid-based pathfinding system using:
- Greedy Best-First Search (GBFS)
- A* Search

The environment supports dynamic obstacle spawning and real-time path re-planning.

## Features
- Adjustable grid size
- Random obstacle generation
- Interactive map editing
- Dynamic obstacle spawning
- Real-time visualization
- Performance metrics (nodes visited, path cost, execution time)

## Installation

1. Clone the repository:
   git clone https://github.com/abdulhanan101/Informed-Searches-Visualization

2. Navigate into the folder:
   cd Informed-Searches-Visualization

3. Create a virtual environment:
   python -m venv venv

4. Activate the environment:
   Windows:
   venv\Scripts\activate

   Linux/Mac:
   source venv/bin/activate

5. Install dependencies:
   pip install -r requirements.txt

## Running the Project

python src/main.py
